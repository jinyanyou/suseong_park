# 대구 수성구 공원 지도

대구광역시 수성구 공원현황 92개소를 카카오맵 위에 표시하고, 동별 번호를 매겨 관리하는 웹 지도입니다.

## 배포

https://jinyanyou.github.io/suseong_park/

## 주요 기능

- 엑셀 원장 기반 92개 공원 데이터 표시
- 카카오맵 주소검색으로 자동 좌표 매핑
- 동/공원명/분류 검색 및 필터
- 현재 목록 CSV 다운로드
- 지도 클릭으로 공원 직접 추가
- GitHub 토큰 연동 시 추가한 공원이 자동으로 저장소에 반영

## 기반 자료

`공원현황(홈페이지 발췌).xlsx` Sheet1 92개 행
